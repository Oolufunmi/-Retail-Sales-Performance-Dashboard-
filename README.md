# 📊 Retail Sales Performance Dashboard | Power BI

## 📌 Project Overview
This project presents an executive-style Power BI dashboard designed to analyze retail sales performance, revenue growth, product mix, bread size adoption, and regional sales distribution.

The dashboard transforms raw retail sales data into interactive business insights that support performance monitoring, operational analysis, and data-driven decision-making.

---

## 🎯 Business Objectives
This dashboard helps answer key business questions such as:

- How are sales performing across different periods?
- Which bread sizes contribute the most revenue?
- Which regions and states generate the highest sales?
- How is sales performance changing month-over-month?
- What percentage of revenue comes from each product category?

---

## 📈 Dashboard Features

### Executive KPI Reporting
- Total Sales
- Units Sold
- Average Order Value
- XL Adoption Rate

### Sales & Revenue Analysis
- Monthly Sales Trend by Year
- Month-over-Month (MoM) Growth Tracking
- Revenue Contribution & Cumulative Share Analysis

### Product & Category Insights
- Bread Size Performance Analysis
- Product Mix Contribution Analysis

### Geographic Analysis
- Geographic Sales Distribution
- Top States by Revenue

### Interactive Features
- Dynamic filtering by:
  - Year
  - Bread Size
  - Town
  - State
- Interactive drilldowns and filtering
